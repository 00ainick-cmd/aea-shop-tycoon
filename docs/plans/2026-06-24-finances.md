# Finances ("The Books") Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Make the money real and felt: an honest line-item invoice, annual AEA dues with a yearly "Books Day," richer unexpected job hits, a line of credit recovery path, a read-only Books page, and corrected dealer/training/apprenticeship economics.

**Tech Stack:** One self-contained HTML file, vanilla JS, no build step, no localStorage. Headless Edge for render verification, in-page `?test=1` harness for logic.
